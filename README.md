# CS Learning

CS Learning is a long-term study project focused on computer science, artificial intelligence, and embodied intelligence. It is designed to document a systematic learning journey across foundational courses, frontier research topics, hands-on implementations, and personal reflections.

The project will gradually cover areas such as computer vision, reinforcement learning, embodied AI and robotics, diffusion models, vision-language-action models, world models, and video generation. Each topic will be studied through a combination of theory, papers, coding practice, experiments, and course projects.

The purpose of this repository is to make learning visible and accumulative. Instead of treating each course or experiment as an isolated task, CS Learning aims to connect them into a coherent knowledge system that can be reviewed, extended, and shared over time.

## Focus Areas

- Computer Vision
- Reinforcement Learning
- Embodied AI and Robotics
- Diffusion Models
- Vision-Language-Action Models
- World Models
- Video Generation

## Project Direction

This repository will grow with the learning process. It will contain notes, implementations, paper summaries, experiment records, and project outcomes as the study plan progresses. The long-term ambition is to build a thoughtful and influential learning archive for modern AI and robotics.
